# 簿記3級
- 会計:
  - 企業の取引・活動(仕入れ、生産、販売、給与支払いなど)を帳簿に記入するルールのこと
  - その帳簿を世間一般の人(銀行、税務署、株主など)に見せることとそのルール
  - 企業内での様々な管理の為に用いられる数値データを計算することとそのルール
- 簿記: 会計を帳簿に付けること
  - 家系簿は他人に見せることがないので適当でも良いが企業の帳簿は他人に見せることがあるので、何がどう正しいのか全員が共通して認識出来るようにルールを定めている

## 複式簿記
- 家系簿はだいだい入出金の金額+何に使ったのか、何から受け取ったのかで付ける
- しかし企業の帳簿は(勘定科目+金額)×2で付ける、これを複式簿記という
- 左右でひと組になっており、左側が「借方」、右側が「貸方」と呼ばれる

- ステップ
  1. どの勘定科目に分類されるのか
  2. その勘定科目が増加してるのか、減少してるのか
  3. 借方、貸方どちらに入れるのか
- その勘定科目が増加した時は借方に、減少した時は貸方に
- ただし負債が減少した場合は借方に、負債が増えた時は貸方に

例:
```
100万円の土地を現金で買った
借方には、資産(土地)、100万円増加(100万円の価値の土地が増えてるので)
貸方には、資産(現金)、100万円の減少(現金で支払っているので)

仕訳: 借方、貸方をセットにきたもの
例: (借) 土地 1,000,000 (貸) 現金 1,000,000

借方と貸方は必ず金額が一致していること
```

- 実務では、、、
  - 伝票を記入してからパソコンなどの画面上で帳簿システム内の伝票・仕訳帳に入力するのが基本。
  - 仕訳することを、仕訳を起こす、伝票を作成する、記帳する、伝票を切るということもある

### 勘定科目
- 資産、負債、純資産、費用、収益
  - 資産=負債+純資産
- 資産: 財産や権利
  - 現金、預金、債権(売掛金、未収入金など)、商品、株式、土地、建物、備品(机、パソコンなど)
- 売掛金: 商品をツケで払って、まだ代金を回収していない状態
  - ツケ代金を請求できる権利、債権のひとつ
- 未収入金: 債権のひとつ、商品以外のものを販売したときのツケ代金を回収できる権利、不動産や有価証券の売却代金など、本業以外の取引で発生したもの

- 負債: マイナスの財産や義務
  - 借入金や買掛金、未払い金などの負債
- 買掛金: 商品をツケで購入し、代金をまだ支払っていない段階のツケを支払う必要がある義務、債務のひとつ
- 未払金: 債務のひとつ、商品以外のものを購入したときのツケ代金を支払う義務、不動産や有価証券の購入代金など、本業以外の取引で発生したもの

- 費用: 経費、家賃の支払い、光熱費、通信費、給料など
- 収益: 売上高、手数料収入、配当収入など

### 商品(本業)か商品以外(本業以外)かが大事
- 商品(本業)か商品以外(本業以外)かで勘定科目が変わるため
- 商品の売買の記帳方法は2種類、分記法と三分法(分記法は試験には出ないし、あまり使わないが商品以外の仕訳でも基本となるので大事)

#### 分記法
- 商品の売買を商品と商品売買益に分けて記帳する方法

例:
```
9月1日 商品を1個80円で購入し、現金で支払った。
9月8日 上記商品を100円で販売し、現金を受け取った。

9/1 (借) 商品 80 (貸) 現金 80

9/8 (借) 現金 100 (貸) 商品 80
                  (貸) 商品売買費 80
```

#### 三分法
- 商品の売買を仕入、売上、繰越商品の3つに分けて記帳する方法

例:
```
9月1日 商品を1個80円で購入し、現金で支払った。
9月8日 上記商品を100円で販売し、現金を受け取った。

9/1 (借) 商品 80 (貸) 現金 80
9/8 (借) 現金 100 (貸) 売上 100
```
- 三文法の方が分岐法よりも優れているとされる点。
  - それは、商品の原価を把握しておく必要がない点。
  - 商品の個数や仕入日によって金額が変わる場合があり、すべて管理しておくのは非常に大変。

## 決算
- 決められた1年間の間にいくら儲けたのか、決算日に資産や負債をいくら持っているのか把握するためのイベント
- 期首: 日本では4月1日
- 期末日: 決算日とも呼ぶ、日本では3月31日
- 会社の場合は4/1～3/31だが個人商店の場合は1/1～12/31の場合がある。(所得税法の関連でそうなる)
  - あと外資系企業では12/31を期末日としている企業が多い。
  - 検定試験では会計期間がいつからいつまでなのかきちんと確認する必要がある。
- 当期純利益: 会計期間内で収益累計＞費用累計の場合、利益
- 当期純損失: 会計期間内で収益累計＜費用累計の場合、損益

- 流れ
  1. 期末までの1年間の期中仕訳をいったん集計する
  2. 決算整理仕訳(決算にしか作成されない仕訳のこと)
  3. 決算整理仕訳をおりまぜて仕訳を再集計する
  4. 貸借対照表と損益計算書をつくる

### 財務諸表
- 1年間の仕分けを集計し下記を3つの表を作成する
  - 貸借対照表(たいしゃくたいしょうひょう)
      - 決算日辞典の資産、負債、純資産それぞれの残高を一覧にした表
      - 左右に分かれており、左側(借方)には資産、右側(貸方)には負債と純資産
      - 資産 = 負債 + 純資産 になっている
      - バランスシート(BS)とも呼ぶ
      - ある時点での状態、財政状態を明らかにしている
  - 損益計算書
      - 1年間の間に収益の累計がいくらだったのか、費用の累計はいくらだったのかを一覧表にしたもの
      - 収益と費用との差額が損益になっている
      - PL(pro fit and loss statement)とも呼ばれる 
      - ある期間での業績、経営成績を明らかにしている
  - キャッシュフロー計算書

## 現金
- 手元: 経理部の出納係が管理している金庫内の現金など 手元にある現金(硬貨や紙幣)は現金という勘定科目を使う
- 現金は資産なので、受け取ったら増えるので借方へ、支払ったら減るので貸方へ仕訳する

```
例:
電気代1万円を現金で支払った。
(借) 水道光熱費 10,000 (貸) 現金 10,000

売掛金1万円を現金で回収した。
(借) 現金 10,000 (貸) 売掛金 10,000
```

### 現金ではないけれど勘定科目を現金にするもの
#### すぐにお金に換金できるもの
- 通貨代用証券:
- 配当金領収証: 株式を保有していると貰えるもの、配当金
- 郵便為替証書: 郵便局にもっていくとお金に替えてくれるもの

```
例:
売掛金100円を郵便為替証書で受け取った。
(借) 現金 100 (貸) 売掛金 100
```

#### 現金過不足
毎日、入出金を頻繁に行い、記帳を行い、週に1回や毎日1回現金残高を確認します。
一致していれば問題ないが、一致しない場合もある。
仕訳の金額を間違えたかもしれないし、現金を紛失したかもしれない。
その際、暫定的に現金過不足という勘定科目を使う。

例: 
```
帳簿上の現金残高が10500円あるが、手元には1万円札が1枚しかない。
(借) 現金過不足 500 (貸) 現金 500

もし原因が判明したら。
現金で500円のガス代を支払っていたが、伝票への起票を忘れていた。
(借) 水道光熱費 500 (貸) 現金過不足 500
```

#### 小口現金
窓口での新聞代集金への支払い、営業マンの旅費の支払いなど日々細かい出勤が多々行われる。
これを毎々会計担当者がのみが行っていると面倒なので、一定の金額の現金を支払い担当者などに渡しておいて、その中で日々の支払いをして貰い、一定期間たったら、何のためにいくら使い、いくら残ったのかをまとめて報告する。
その報告を元に仕訳を行い、使われた分を再度補充する。
この流れの中で最初に設定した一定金額の現金のことを小口現金として仕訳する。

例: 
```
月曜日の朝、出納係は用度係(支払担当者)に10万円渡す。
(借) 小口現金 100,000 (貸) 現金 100,000

月曜~金曜の間に様々な支払いをした。
翌週の月曜日に用度係は1週間で支払ったもの全ての領収書を出納係に渡す。
出納係は仕訳を行った。
(借) 水道光熱費 3000 (貸) 小口現金 90,000
(借) 旅費交通費 57,000
(借) 通信費 30,000

減った金額を用度係に渡して現金の補充をした。
(借) 小口現金 90,000 (貸) 現金 90,000
```

## 預金
例: 
```
銀行へ行って現金30,000円を普通預金に預け入れた。
(借) 普通預金 30,000 (貸) 現金 30,000  
現金が普通預金に姿を変えた。

銀行へ行って現金50,000円を当座預金に預け入れた。
(借) 当座預金 50,000 (貸) 現金 50,000

普通預金20,000円を定期預金に振り替えた。その際、手数料として200円が普通預金から差し引かれた。
(借) 定期預金 20,000 (貸) 普通預金 20,200
(借) 支払手数料 200 
「支払手数料」は費用

普通預金に利息1,000円がついた。
(借) 普通預金 1,000 (貸) 受取利息 1,000  
「受取利息」は収益

手元にある3,000円の郵便為替証書を普通預金に預け入れた。
(借) 普通預金 3,000 (貸) 現金 3,000  
郵便為替証書は「現金」扱い
```

### 当座預金
当座預金を開くと、小切手や手形が使えるようになる。

#### 小切手
- 小切手は現金の代わりに支払いに使える。
- 小切手は振り出した(小切手を支払い時に現金の代わりに渡すこと)時点で当座預金が減少したものとして考える。
  - つまり、仕訳時は貸方に「当座預金」と記帳する。
- 逆に小切手を受け取った側は相手から小切手を受け取った時点で「現金」をもらったものと考える。
  - 借方に「現金」と記帳する

例: 
```
Ａ商店はＢ商店（パソコンショップ）から、Ａ商店の事務所内で使用するためにパソコンを20万円で購入し、小切手に金額を記入してＢ商店に渡した。  Ａ商店側の仕訳は: 
(借) 備品 200,000 (貸) 当座預金 200,000  

Ｂ商店側の仕訳は  
(借) 現金 200,000 (貸) 売上 200,000
```

- 自己振出小切手の場合は減った当座預金をそのまま減らなかった扱いにする
- 他人からもらった小切手で支払いをする場合は現金が減った扱いになる

例:
```
上のパソコンの支払いの続きでB商店はC商店の商品をA商店から受け取った小切手で支払いをした。
B商店の仕分け
(借) 備品 200,000 (貸) 現金 200,000  

C商店側の仕訳は  
(借) 現金 200,000 (貸) 売上 200,000

その後C商店はA商店の商品をB商店から受け取った小切手で支払った。
C商店の仕分け
(借) 備品 200,000 (貸) 現金 200,000  

A商店側の仕訳は  
(借) 当座預金 200,000 (貸) 売上 200,000
```

#### 当座借越
- 当座借越契約: 銀行に予めいくらまで条件(審査)なしでお金を借りられるという契約を交わすこと。
- 当座借越契約によってお金を借りることで当座預金残高がマイナスになることを当座借越という。

例: 
```
買掛金400,000の支払いのために、小切手400,000を降り出した
(当座預金は50,000しかないが、予め銀行と限度額1,000,000の当座借越契約を締結している)

(借) 買掛金 400,000 (貸) 当座預金 400,000

期中の勘定はこのままにしておき、期末に当座借越として整理を行うことになる。
```

#### 口座を複数開設している場合
例: 
```
A銀行の当座預金とB銀行の当座預金に100円ずつ預金した。
(借) 当座預金 A銀行 100 (貸) 200
(借) 当座預金 B銀行 100

預金口座ごとに勘定科目を設定していない場合
この場合は当座預金 200の明細を別途、補助簿に明細を作る必要がある
(借) 当座預金 200 (貸) 現金 200

B銀行の普通預金300円からA銀行の当座預金に移した。
(借) 当座預金 A銀行 300 (貸) 普通預金 B銀行 300
```

- 商品以外の売買であれば分岐法+未払い金、未収入金

### 付随費用
- 簿記では仕入に掛かった費用を仕入に算入する。
- 諸費用は仕入れた物の取得原価に入れる。

例:
```
商品200円を掛けで仕入れ、引取手数料として現金10円を払った(当該手数料は買主負担)
(借) 仕入 210 (貸) 買掛金 200
              (貸) 現金 10

売主負担の場合
商品300円を掛けで販売し、発送費20円(売主負担)を現金で支払った。

(借) 売掛金 300 (貸) 売上 300
(借) 発送費 20 (貸) 現金 20

300円の商品を掛けで仕入れ、発送費20円(売主負担)は買主が立て替えて現金で支払った。

立替金を使う場合
(借) 仕入 300 (貸) 買掛金 300
(借) 立替金 20 (貸) 現金 20

買掛金を減額する場合
(借) 仕入 300 (貸) 買掛金 280
                        (貸) 現金 20

商品100円を掛けで売った。買主負担の送料10円を売主が現金で支払った。なおその送料分を売掛金及び、売上に上乗せした。
(借) 売掛金 110 (貸) 110
(借) 発送費 10 (貸) 現金10
```

### 原価・有高の算定
- 商品有高や原価の算定方法にはいくつかあり、代表的なのが下記２つ。

例:
```
5月1日、単価100円の商品を10個仕入れた。
5月10日、単価130円の商品を20個仕入れた。
5月15日、商品1個を150円で販売した。
5月15日に販売した商品の仕入れ値はいくらとするのか？

先入先出法(さきいれさきだしほう)
の場合は100円

移動平均法
の場合は120円
(100円×10個+130円×20個)/30個=120円/個
```


## 手形
- 代金の支払いに用いられたり、お金を貸す時の借用書代わりとして使われる
- 種類
  - 約束手形
    - 2者間で使われる
  - 為替手形(簿記3級ではでない)
    - 3者間で使われる
- 手形の使われ方
  1. A商店が商品をB商店から仕入れた時、手形(約束手形)を作成きて、B商店に渡す。
  2. B商店はこの手形を銀行に預ける
  3. この手形がサイト90日(手形に記載されている期限)の手形であれば作成日から90日後に入金される。つまりA商店の預金口座から手形代金が引き落とされ(決済され)、B商店の預金口座に振り返られる。

例:
```
1. A商店はB商店より商品5000円を仕入れ、代金は手形を作成して渡した。

A商店: (借) 仕入 5000 (貸) 支払手形 5000
B商店: (借) 受取手形 5000 (貸) 5000

支払手形: 負債(債務)
受取手形: 資産(債権)

2. B商店は手形を銀行に預けた

仕分なし

手形はすぐ銀行に持っていく為、基本的にこれは問題に出ない

3. 手形の決済期日になりB商店の当座預金口座に手形代金が入金された。

A商店: (借) 支払手形 5000 (貸) 当座預金 5000
B商店: (借) 当座預金 5000 (貸) 受取手形 5000

4. A商店がB商店から現金1万円を借り、手形をB商店に渡した。

A商店: (借) 現金 10,000 (貸) 手形借入金 10,000
B商店: (借) 手形貸付金 10,000 (貸) 現金 10,000

手形を使わない場合は、貸付金、借入金になる
```

## 電子記録債権、債務
- 電子記録債権・債務は手形のめんとうな事務手続きの代わりとなるように使われるようになった
- なので仕訳では基本的に手形の部分を電子記録に代えるだけ
- 取引を行った時に電子記録債権・債務がいきなり発生するのではなくまず売掛金、買掛金が発生し、変更するという流れ
- 売った側から電子債権記録機関に発生記録の請求(登録)を行うこともできるが、その場合は買った側の承諾が必要になるので、基本的には買った側が記録の請求を行う


例:
```
A社側の仕訳
8/20にA社はB社に対して、商品500円を掛けで販売した。

(借) 売掛金 500 (貸) 売上 500

9/30にB社はA社に対する買掛金について、電子債権記録機関に債務発生の記録を行った。(ここで売掛金が電子記録債権に変わる)

(借) 電子記録債権 500 (貸) 売掛金 500

1/4に電子記録債権の支払いが行われた。(B社の当座預金からA社の当座預金に資金が振り返られた)

(借) 当座預金 500 (貸) 電子記録債権 500

B社側
(借) 仕入 500 (貸) 買掛金 500

(借) 買掛金 500 (貸) 電子記録債務 500

(借) 電子記録債務 500 (貸) 当座預金 500
```

## 経費

- 逆に少額でなく、換金性の高いもの(郵便切手や収入印紙)は資産として扱い、期中は費用としておいて、期末に未使用分を資産にする

## 貸付金、借入金
- 誰かにお金を貸したら「貸付金(資産)」、お金を借りたら「借入金(負債)」を計上する

例:
```
AはBに現金100円を年利2%、1年後に一括返済で貸し付け、利息も元金返済時に受け取ることになった。

Aの仕訳

(借) 貸付金 100 (貸) 現金 100

Bの仕訳

(借) 現金 100 (貸) 借入金 100

1年後の返済時

Aの仕訳

(借) 現金 102 (貸) 貸付金 100
             (貸) 受取利息 2

Bの仕訳

(借) 借入金 100 (貸) 現金 102
(借) 支払利息 2
```

## 預り金
- 給料を従業員に支払う際さまざまな税金や社会保険料がかかる
- 社会保険料は企業と従業員で半分ずつ支払う
- 所得税は従業員が全額負担
- 社会保険料と所得税を支払い作業は予め企業が従業員の給料から差し引いて(源泉徴収)、翌月に税務署や年金事務所に支払う(納付)
- この源泉徴収したお金を預り金(負債)として仕訳する
  - 社会保険料預り金(健康保険料、厚生年金保険料、雇用保険料)
  - 所得税預り金

例:
```
給料計算期間が9/1～10/20の10/28払いの給与総額30万円で所得税2万円、社会保険料3万円の場合、企業負担分(法廷福利費)は4万円とする。

10/28の給与支払い時

(借) 給料 300,000 (貸) 当座預金 250,000
                 (貸) 所得税預り金 20,000
                 (貸) 社会保険料預り金 30,000

11月納付時

(借) 所得税預り金 20,000 (貸) 当座預金 90,000
(借) 社会保険料預り金 30,000
(借) 法廷福利費 40,000
```

## 仮払金、仮受金
- 仮払金: 従業員が出張をする際の費用として一時的に渡すものなど
- 仮受金: 入金されたものが何の入金か分からない際に一時的に仕分けておくなど

例:
```
たとえば従業員が明日出張するので、本日、その従業員に出張の諸費用に充てるために現金2万円を渡した

(借) 仮払金 20,000 (貸) 現金 20,000

後日、従業員が出張から帰ってきて、諸費用（電車代13,000円、タクシー代2,000円）の領収書と、残った現金を経理係に渡した

(借) 旅費交通費 15,000 (貸) 仮払金 20,000
(借) 現金 5,000

自商店用に利用しているICカードに現金3万円をチャージした。なおチャージしたときには「仮払金」勘定を用いている

(借) 仮払金 30,000 (貸) 現金 30,000

従業員が出張し、上記ICカードを用いて電車代2,000円を支払った。  

(借) 旅費交通費 2,000 (貸) 仮払金 2,000

出張費用がいくらかかるのかは事前にはわからないため、事前にまとまったお金を渡しておいて、後で上記ICカードに2,000円を補充して残高を3万円に戻して、小口現金的な使い方をしている

もう1つICカードにチャージした時点で費用にしてしまう方法もある
自商店用に利用しているICカードに現金3万円をチャージした。なお当該ICカードは出張の旅費交通費にしか使わないため、チャージしたときに費用処理をしている。  

(借) 旅費交通費 30,000 (貸) 現金 30,000

従業員が出張し、上記ICカードを用いて電車代2,000円を支払った。

仕訳なし

入金はなされても、何の代金なのかが不明な場合には、「仮受金」勘定を計上しておく場合がある。
たとえば、本日当座預金口座に1,000円が入金されたが、何の代金なのか不明であるため、いったん仮受金に計上しておく。  

(借) 当座預金 1,000 (貸) 仮受金 1,000

後日、先日の入金はＢ商店に対する売掛金が回収されたものであることが判明した

(借) 仮受金 1,000 (貸) 売掛金 1,000
```
